Graphical editor cw2

  • Автор: Ильин Егор, студент 1 курса СПбГЭТУ "ЛЭТИ", Факультет Компьютерных Технологий и Информатики, группа 2303
  • Данная программа разработана в рамках курсовой работы по программированию во втором семестре, откуда название cw2 (course-work-semester2)
  • Использованные ресурсы:
    • Язык программирования: C++ 20 (однако большая часть кода будет работать и на старой версии вроде C++ 14)
    • IDE: Microsoft Visual Studio 2022 Community
    • Изображения и иконки: GIMP 2.10.24
    • Использованные библиотеки:
      1. Qt 6.5.0 для графического интерфейса
      2. libpng для работы с png файлами
    • Сайты использованные во время разработки:
      1. wordtohtml.net - создание html файлов на основе word, использовалось для создания данной справки
      2. www.flaticon.com - сайт с иконками в свободном доступе
      3. Оффициальные документации по Qt, libpng, ряд других сайтов (stackoverflow, wikipedia, github, и т.д.)
