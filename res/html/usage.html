General information on usage

  • The Graphical editor cw2 allows to edit png images. It can open png files in any configuration, but all the work with loaded image and its further saving to a file would go under these parameters:
    • Color scheme RGBA with 8-bit channel width (thus each component can take values in range 0-255).
    • As of the current version, no additional png data (tIME. eXIF and alike) is used.
  • The possibility of working with other image formats (namely jpeg and bmp) is partially provided for in the code, but not currently supported.
  • Note that due to the program being developed for a short while (~10 days in total), it lacks a lot of common instrumentarium which can possibly be added in the future.
  • The program is free to use and distribute in any possible way and the author is not responsible for any consequences it would lead it.
