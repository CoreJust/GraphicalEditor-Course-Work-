Graphical editor cw2

  • Author: Yegor Ilyin, student of ETU, 1st course, Faculty of Computer Technologies and Informatics, group 2303
  • This program is made as a course work in second semester, thus its name cw2 (course-work-semester2)
  • Used resources:
    • Language: C++ 20 (although almost all the code would work with old versions such as C++ 14)
    • IDE: Microsoft Visual Studio 2022 Community
    • Pictures and icons: GIMP 2.10.24
    • Libraries used in code:
      1. Qt 6.5.0 for GUI
      2. libpng for handling png image files
    • Services used during development:
      1. wordtohtml.net - site for creating html files from word, used for this help
      2. www.flaticon.com - site with publicly available icons
      3. Qt, libpng official documentations, some other sites (stackoverflow, wikipedia, github, etc)
