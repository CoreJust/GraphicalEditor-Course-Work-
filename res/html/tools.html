Tools

  • General - no matter the chosen instrument, you can access the menus and zoom the image.
  • Moving - the first tool in the toolset. Allows to move around the image.
  • Pen - the second tool in the toolset. Allows to draw curves on the image.
  • Color-picker - allows to pick a color on the image.
  • Color - the color label displaying the current color. On clicking opens color choice menu.
  • Width slider - allows to set the width of the current tool. Disappears if the current tool doesn't have width.
