Menus

  1. File - this menu contains several options:
    1. New - allows to create a new image with a certain size and filled with a certain color.
    2. Open - allows to open an image from a chosen file.
    3. Save - allows to save the current image to the file from where it was loaded. In case of a new image, it acts the same as "Save as".
    4. Save as - allows to save the image to an optional file.
    5. Exit - allows to quit the program.
  2. Tools - some tools that are used with a set of settings (thus not in the image edit scene):
    1. Filter by channel - allows to replace the chosen channel of all the pixels in the image to some value.
    2. Split image into subimages - allows to split the image into NxM parts and save each to the chosen path.
    3. Split image with grid - allows to split the image into NxM parts with a grid, which is a set of vertical and horizontal lines with a certain line width and color.
    4. Draw circle - allows to draw a circle with the chosen center point, radius, line width, color, and (optionally) filled with some color.
    5. Draw rectangle - allows to draw a rectangle and its diagonals with the chosen left-top point, side length, line width, color, and (optionally) filled with some color.
  3. Image - shows current image's info.
  4. Help - shows a help window (yes, the one you are reading now).
