Меню

  1. File - это меню содержит несколько опций:
    1. New - создаёт новый файл с выбранными размером и цветом.
    2. Open - открывает выбранное изображение (из файла).
    3. Save - сохраняте изображение в тот же файл, откуда оно было загружено. Для новых файлов действует как "Save as".
    4. Save as - сохраняет изображение в выбранный файл.
    5. Exit - выход из программы.
  2. Tools - инструменты, требующие ряда настроек (и потому не используемые непосредственно в сцене редактора):
    1. Filter by channel - позволяет заменить у всех пикселей указанный канал на выбранное значение.
    2. Split image into subimages - разбивает изображение на NxM частей, каждую из которых сохраняет по указанному пути.
    3. Split image with grid - разбивает изображение на NxM частей посредством отрисовки решётки, состоящей из горизонтальных и вертикальных линий с указанной толщиной и цветом.
    4. Draw circle - отрисовывает круг с указаным центром, радиусом, толщиной линии, цветом и (опционально) залитый некоторым цветом.
    5. Draw square - отрисовывает квадрат с диагоналями с указаными левой-верхней точкой, стороной, толщиной линии, цветом и (опционально) залитый некоторым цветом.
  3. Image - отображает информацию о текущем изображении.
  4. Help - открывает справке (да, ту самую, которую вы сейчас читаете).
